## E-Commerce Sales Analysis using SQL and Tableau

## Project Overview

In this project, I took a deep dive into real-world e-commerce sales data, using both SQL and Tableau to uncover actionable insights for the business.

##  Features

* Data cleaning using SQL
* Complex joins across multiple tables
* Revenue and business insights
* Advanced SQL queries (window functions)
* Interactive Tableau dashboard

##  Dataset

* Customers
* Products
* Orders
* Order Items

##  Key Insights

* Total Revenue Analysis
* Top Selling Products
* Customer Lifetime Value
* Monthly Sales Trends
* Sales by Country

##  Tools Used

* SQL (Microsoft SQL)
* Tableau
* Excel
